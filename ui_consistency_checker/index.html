How Consistency Is Scored

Each dimension is normalized to a 0–100 scale and averaged into a final consistency score.

  • Colors: score drops 15 pts for each color above 5
  • Fonts: score drops 20 pts for each font style above 3
  • Spacing: full score for 4px or 8px grid; reduced for irregular values
  • Components: direct 0–100 self-assessment of style uniformity
